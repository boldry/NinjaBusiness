# Project Evaluation Checklist

## Pre-Evaluation Setup
- [ ] **Project Information Collected**
  - [ ] Project name and description
  - [ ] Key stakeholders identified
  - [ ] Initial market research completed
  - [ ] Competitive landscape mapped

- [ ] **Team Assignment**
  - [ ] Product Manager assigned
  - [ ] Technical Lead assigned
  - [ ] Business Lead assigned
  - [ ] Executive Sponsor assigned

### Market Research
- [ ] **TAM/SAM/SOM Analysis**
  - [ ] Total Addressable Market calculated
  - [ ] Serviceable Addressable Market defined
  - [ ] Serviceable Obtainable Market estimated
  - [ ] Market growth rate researched
  - [ ] Data sources documented

- [ ] **Competitive Analysis**
  - [ ] Direct competitors identified
  - [ ] Indirect competitors identified
  - [ ] Competitive positioning analyzed
  - [ ] Competitive advantages identified
  - [ ] Market gaps identified

- [ ] **Customer Research**
  - [ ] Target customer segments defined
  - [ ] Customer personas created
  - [ ] Customer pain points validated
  - [ ] Customer interviews conducted
  - [ ] Market demand validated

### Business Case
- [ ] **Value Proposition**
  - [ ] Problem statement clearly defined
  - [ ] Solution approach outlined
  - [ ] Unique value proposition identified
  - [ ] Competitive differentiation established

- [ ] **Revenue Model**
  - [ ] Revenue streams identified

  - [ ] Revenue projections created

- [ ] **Financial Projections**
  - [ ] Development costs estimated
  - [ ] Operating costs estimated
  - [ ] Revenue projections created
  - [ ] Break-even analysis completed
  - [ ] ROI calculations done

### Strategic Alignment
- [ ] **Corporate Objectives**
  - [ ] Aligns with company vision
  - [ ] Supports strategic goals
  - [ ] Fits technology roadmap
  - [ ] Leverages existing capabilities

- [ ] **Resource Requirements**
  - [ ] Team requirements identified
  - [ ] Technology requirements identified
  - [ ] Budget requirements estimated
  - [ ] Timeline requirements estimated

### Risk Assessment
- [ ] **Market Risks**
  - [ ] Market size risks identified

- [ ] **Business Risks**
